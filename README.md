# AR Commerce - Try Before You Buy

A modern, single-page landing website showcasing WebAR (Web-based Augmented Reality) technology for e-commerce. Experience products in your space before making a purchase with this interactive AR shopping platform.

AR Commerce is a cutting-edge landing page that demonstrates the future of online shopping through WebAR technology. The website features a sleek, futuristic design with holographic effects, smooth animations, and an interactive AR experience that allows users to visualize products in their environment using their device's camera.

The project showcases how AR can revolutionize e-commerce by enabling customers to "try before they buy" directly from their web browser, without requiring any app downloads.

## üöÄ Setup Instructions

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, but recommended for camera access)
- A device with a camera (for AR functionality)

### Installation

1. **Clone or Download** the project files to your local machine

2. **Open the Project**
   - Option A: Open `index.html` directly in your browser (note: camera features may be limited due to browser security)
   - Option B: Use a local web server for full functionality:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```

3. **Access the Website**
   - Navigate to `http://localhost:8000` (or your chosen port)
   - Or simply open `index.html` in your browser

4. **Test AR Features**
   - Click "Start AR Experience" or "Try in AR" buttons
   - Allow camera permissions when prompted
   - Point your device at a flat surface to see AR effects

### Browser Compatibility

- **Chrome/Edge**: Full support (recommended)
- **Firefox**: Full support
- **Safari**: Full support (iOS 11+)
- **Mobile Browsers**: Works on iOS Safari and Chrome Mobile
